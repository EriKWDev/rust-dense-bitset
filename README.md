# rust-dense-bitset

Efficient and flexible bitset rust library. 


For large (but sparse) bitsets the [hibitset](https://github.com/slide-rs/hibitset) library can be considered instead.

## Running the tests

Run the tests with

```
cargo test
```

## Documentation

Generate the documentation with

```
cargo doc
```
